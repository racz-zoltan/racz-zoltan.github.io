Desktop & Mobile
Install the app

On every device,
no app store needed.

CarryPass is a Progressive Web App. Install it in seconds directly from your browser — it runs like a native app with no download required.

Safari
iPhone & iPad · Required
  1. 1Open CarryPass in Safari — the default iOS browser.
  2. 2Tap the Share button in the bottom toolbar.
  3. 3Scroll down and tap "Add to Home Screen".
  4. 4Confirm the name and tap Add — done.
Chrome & Edge
iPhone & iPad · Supported
  1. 1Open CarryPass in Chrome or Edge on iOS.
  2. 2Tap the Share button in the bottom toolbar.
  3. 3Tap "Add to Home Screen".
  4. 4Confirm the name and tap Add — done.

Firefox for iOS does not support PWA installation. Use Safari, Chrome, or Edge instead.

On iOS 16.4+ the installed app can show local notifications about vault status. CarryPass does not subscribe to any external push service. If notifications are added in a future update, you'll be prompted to allow them.
Chrome
Android · Recommended
  1. 1Open CarryPass in Chrome.
  2. 2A banner may appear at the bottom — tap "Install" or "Add to Home screen".
  3. 3If no banner: tap the ⋮ Menu (top-right) → "Add to Home screen".
  4. 4Tap Install in the confirmation dialog.
Firefox
Android
  1. 1Open CarryPass in Firefox for Android.
  2. 2Tap the ⋯ Menu button.
  3. 3Tap "Install" or "Add to Home screen".
  4. 4Confirm by tapping Add.
Samsung Internet
Android · Samsung devices
  1. 1Open CarryPass in Samsung Internet.
  2. 2Tap the ☰ Menu at the bottom.
  3. 3Tap "Add page to" → "Home screen".
  4. 4Tap Add to confirm.
On Android, Chrome gives the richest install experience including a proper splash screen and standalone window. If you're on a Samsung device, Samsung Internet works equally well.
Chrome
Windows · macOS · Linux
  1. 1Open CarryPass in Chrome.
  2. 2Look for the Install icon on the right side of the address bar.
  3. 3Click it, then click "Install" in the prompt.
  4. 4CarryPass opens as a standalone window — pinnable to your taskbar.
Microsoft Edge
Windows · macOS
  1. 1Open CarryPass in Edge.
  2. 2Look for the Install app icon on the right side of the address bar.
  3. 3Click it, then click "Install" in the prompt.
  4. 4CarryPass will appear in your Start menu and taskbar.
Safari
macOS Sonoma 14+
  1. 1Open CarryPass in Safari on macOS Sonoma or later.
  2. 2Click File in the menu bar.
  3. 3Select "Add to Dock".
  4. 4Click Add — the app appears in your Dock.
Firefox
Desktop — Not supported

Firefox on desktop does not support PWA installation. Use Chrome or Edge for the best desktop install experience.

Once installed, CarryPass on desktop gets its own window, taskbar/Dock entry, and runs independently of the browser. It feels and behaves like a native app.

Use a strong passphrase: at least 16 random characters, or preferably a long multi-word phrase.

Select 12 icons across 3 short rounds. The icon positions change, so remember the icons and their order, not their location.

What is CarryPass?

CarryPass is an open-source, client-side password manager and team credential-sharing tool. It runs in the browser as an offline-first Progressive Web App and uses strong cryptographic primitives to generate passwords and protect encrypted vaults without requiring accounts, telemetry, or server-side access to secrets.
