Uncompromising Security by Design

CarryPass doesn’t cut corners for scalability — it doesn’t need to. All encryption happens locally on your device, allowing the application to use strong, resource-intensive cryptographic methods without worrying about server cost or performance tradeoffs.

Services
This list contains your custom password generation settings, organized by service name. When you select a service from the list, its settings are automatically applied. You can export and import these settings as an encrypted file.

    Service
    The site or app name that you want to access. The service name affects password generation consistency. All service names are converted to lowercase during normalization.
    Master Password
    Your main password. Never shared or stored, long and complex. Used to derive all your secure passwords. All your generated passwords are as secure as long and complex this one Master Password is!

    Generating passwords ...

    Character types
    These define what kind of characters are allowed in your generated password. Separator is for readability, also counts as symbol.
    Length
    Desired length of the generated password. More is stronger.
    Security
    Controls computation rounds and memory usage. Higher number = more secure passwords, but slower generation. Performance thresholds at 33,333 and 66,666.
    Save Settings
    Toggle 'Save Settings' before password generation and your custom settings are saved to local storage. You can manage them under 'Services'.
    Username
    If a service requires a username, separate from your email address, you may use this.

    Additional Passwords
    Additional passwords allow for password rotation if a service requires it.
