Manage Settings
Lists your custom password generation settings by service name. Default settings automatically adjust when a service is chosen from this list. Settings can be exported and imported in an encrypted file.

    Service
    The site or app name that you want to access. The service name affects password generation consistency.
    Master Password
    Your main password. Never shared or stored, long and complex. Used to derive all your secure passwords. All your generated passwords are as secure as long and complex this one Master Password is!

    Generating passwords ...

    Settings
    These define what kind of characters are allowed in your generated password. Separator is for readability, also counts as symbol.
    Length
    Desired length of the generated password. More is stronger.
    Password Security
    Controls computation rounds and memory usage. Higher number = more secure passwords, but slower generation.
    Store Settings
    Toggle 'Store Settings' before password generation and your custom settings are saved to local storage. You can manage them under 'Manage Settings'.
    Username
    If a service requires a username, separate from your email address, you may use this.

    Additional Passwords
    Additional passwords allow for password rotation if a service requires it.
